# Plan A: Foundation + Ghostty Proof-of-Concept Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A runnable Swift macOS app that opens a single window containing one embedded Ghostty terminal pane running the user's default shell — proving the GhosttyKit-in-SwiftUI integration works end-to-end before we layer on projects, tabs, and inspector UI.

**Architecture:** Swift + SwiftUI host with an `NSViewRepresentable` wrapper around a `CAMetalLayer`-backed `NSView`. GhosttyKit.xcframework is vendored as a binary framework, fetched by a checksum-pinned shell script (the cmux pattern). We follow cmux's `Sources/GhosttyTerminalView.swift` as the reference implementation but implement only the **minimum viable subset** of `ghostty_*` calls needed for one working terminal.

**Tech Stack:**
- Swift 5.10, Xcode 16
- SwiftUI (window/scene), AppKit (`NSViewRepresentable`, `NSView`, `CAMetalLayer`)
- GhosttyKit.xcframework (binary, sourced from `manaflow-ai/ghostty` releases)
- Target: macOS 14+, ARM64

**Reference sources (cite these in commits / PR messages):**
- cmux: `https://github.com/manaflow-ai/cmux`
- cmux's `Sources/GhosttyTerminalView.swift` (single 13k-line file containing the entire C bridge + NSView + IME plumbing)
- cmux's `scripts/download-prebuilt-ghosttykit.sh`
- Pinned ghostty SHA at the time of writing: `f7880c47313a8697ee7f58969564772cb70eb6e9`, sha256 `d8647cf2fa1e54912ca781db81871932a4c4becc96b58a3eda4f7e66c979cdbf`

**Success criteria for Plan A:**
- App launches, shows a window with a working terminal pane
- Typing `ls` + Enter lists the home directory
- Arrow-key history works
- Cursor blinks, selection works, scrollback works
- `⌘C` / `⌘V` copy/paste work
- Closing the window quits cleanly with no Metal warnings or zombie PTYs

```bash
#!/usr/bin/env bash
set -euo pipefail

# Fetches a prebuilt GhosttyKit.xcframework from manaflow-ai/ghostty's
# release tarballs, verifies its SHA256 against Scripts/ghosttykit-checksums.txt,
# and unpacks it into Vendor/GhosttyKit.xcframework.

REPO_ROOT="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
CHECKSUMS_FILE="$REPO_ROOT/Scripts/ghosttykit-checksums.txt"
VENDOR_DIR="$REPO_ROOT/Vendor"
FRAMEWORK_DIR="$VENDOR_DIR/GhosttyKit.xcframework"

if [[ ! -f "$CHECKSUMS_FILE" ]]; then
    echo "error: $CHECKSUMS_FILE not found" >&2
    exit 1
fi

# Last entry in the checksums file is the active pin
read -r GHOSTTY_SHA EXPECTED_SHA256 <<<"$(tail -n 1 "$CHECKSUMS_FILE")"

if [[ -z "${GHOSTTY_SHA:-}" || -z "${EXPECTED_SHA256:-}" ]]; then
    echo "error: could not parse pin from $CHECKSUMS_FILE" >&2
    exit 1
fi

if [[ -d "$FRAMEWORK_DIR" ]]; then
    PIN_MARKER="$FRAMEWORK_DIR/.ghostty-sha"
    if [[ -f "$PIN_MARKER" ]] && [[ "$(cat "$PIN_MARKER")" == "$GHOSTTY_SHA" ]]; then
        echo "GhosttyKit.xcframework already at $GHOSTTY_SHA — skipping."
        exit 0
    fi
fi

mkdir -p "$VENDOR_DIR"
TMP_DIR="$(mktemp -d)"
trap 'rm -rf "$TMP_DIR"' EXIT

ARCHIVE_NAME="GhosttyKit.xcframework.tar.gz"
DOWNLOAD_URL="https://github.com/manaflow-ai/ghostty/releases/download/xcframework-${GHOSTTY_SHA}/${ARCHIVE_NAME}"
ARCHIVE_PATH="$TMP_DIR/$ARCHIVE_NAME"

echo "Downloading $DOWNLOAD_URL ..."
curl --fail --location --retry 10 --retry-delay 5 -o "$ARCHIVE_PATH" "$DOWNLOAD_URL"

ACTUAL_SHA256="$(shasum -a 256 "$ARCHIVE_PATH" | awk '{print $1}')"
if [[ "$ACTUAL_SHA256" != "$EXPECTED_SHA256" ]]; then
    echo "error: SHA256 mismatch" >&2
    echo "  expected: $EXPECTED_SHA256" >&2
    echo "  actual:   $ACTUAL_SHA256"   >&2
    exit 1
fi

rm -rf "$FRAMEWORK_DIR"
tar --no-same-owner -xzf "$ARCHIVE_PATH" -C "$TMP_DIR"
mv "$TMP_DIR/GhosttyKit.xcframework" "$VENDOR_DIR/"
echo "$GHOSTTY_SHA" > "$FRAMEWORK_DIR/.ghostty-sha"

echo "GhosttyKit.xcframework installed at $FRAMEWORK_DIR (pinned to $GHOSTTY_SHA)."
```

## Task 4: GhosttyApp singleton (app_t lifecycle)

**Files:**
- Create: `AgenticIDE/Terminal/GhosttyApp.swift`
- Modify: `AgenticIDE/AgenticIDEApp.swift`

Implements the **app initialization** sequence from cmux's `Sources/GhosttyTerminalView.swift` lines 1780–1975: `ghostty_init` → `ghostty_config_new` → load defaults → `ghostty_config_finalize` → fill `ghostty_runtime_config_s` → `ghostty_app_new`. The singleton owns the `ghostty_app_t` for the whole process. We never call `ghostty_app_free` (cmux doesn't either; the app lives until process exit).

- [ ] **Step 1: Create GhosttyApp.swift**

Write `/Users/fabio/Documents/playground/Agentic-IDE/AgenticIDE/Terminal/GhosttyApp.swift`:

```swift
import Foundation
import AppKit

/// App-wide singleton wrapping the ghostty_app_t handle.
/// Created once at process start, never freed.
@MainActor
final class GhosttyApp {
    static let shared = GhosttyApp()

    private(set) var app: OpaquePointer?
    private var config: OpaquePointer?

    private init() {}

    /// Performs ghostty_init + ghostty_config_new + ghostty_app_new.
    /// Call once from AgenticIDEApp.init.
    func bootstrap() {
        guard app == nil else { return }

        let initResult = ghostty_init(UInt(CommandLine.argc), CommandLine.unsafeArgv)
        guard initResult == GHOSTTY_SUCCESS else {
            print("ghostty_init failed: \(initResult.rawValue)")
            return
        }

        guard let cfg = ghostty_config_new() else {
            print("ghostty_config_new returned nil")
            return
        }

        ghostty_config_load_default_files(cfg)
        ghostty_config_finalize(cfg)

        var runtimeConfig = ghostty_runtime_config_s()
        runtimeConfig.userdata = Unmanaged.passUnretained(self).toOpaque()
        runtimeConfig.supports_selection_clipboard = false

        runtimeConfig.wakeup_cb = { _ in
            // Ghostty wants the main thread to call ghostty_app_tick soon.
            DispatchQueue.main.async {
                if let app = GhosttyApp.shared.app {
                    ghostty_app_tick(app)
                }
            }
        }

        // Minimal action callback: ignore everything for now, return false
        // so Ghostty knows we didn't handle it.
        runtimeConfig.action_cb = { _, _, _ in
            return false
        }

        guard let appHandle = ghostty_app_new(&runtimeConfig, cfg) else {
            print("ghostty_app_new returned nil")
            ghostty_config_free(cfg)
            return
        }

        self.app = appHandle
        self.config = cfg
        print("GhosttyApp bootstrapped.")
    }
}
```

```swift
import AppKit
import QuartzCore

/// NSView whose backing layer is a CAMetalLayer that Ghostty's renderer draws into.
/// Owns no `ghostty_surface_t` directly — that's GhosttySurface's job. This view
/// is just the window-system surface (Metal layer + event entry point).
final class GhosttyNSView: NSView {
    /// Set after the surface has been created. Used to forward input events.
    weak var surface: GhosttySurface?

    override init(frame frameRect: NSRect) {
        super.init(frame: frameRect)
        wantsLayer = true
        layer?.masksToBounds = true
    }

    required init?(coder: NSCoder) { fatalError("init(coder:) not used") }

    override func makeBackingLayer() -> CALayer {
        let metalLayer = CAMetalLayer()
        metalLayer.pixelFormat = .bgra8Unorm
        metalLayer.isOpaque = false
        metalLayer.framebufferOnly = false
        return metalLayer
    }

    override var acceptsFirstResponder: Bool { true }

    override func becomeFirstResponder() -> Bool {
        let ok = super.becomeFirstResponder()
        if ok, let s = surface?.handle {
            ghostty_surface_set_focus(s, true)
        }
        return ok
    }

    override func resignFirstResponder() -> Bool {
        let ok = super.resignFirstResponder()
        if ok, let s = surface?.handle {
            ghostty_surface_set_focus(s, false)
        }
        return ok
    }

    override func setFrameSize(_ newSize: NSSize) {
        super.setFrameSize(newSize)
        guard let s = surface?.handle else { return }
        let scale = window?.backingScaleFactor ?? 2.0
        ghostty_surface_set_content_scale(s, scale, scale)
        let widthPx = UInt32(newSize.width  * scale)
        let heightPx = UInt32(newSize.height * scale)
        ghostty_surface_set_size(s, widthPx, heightPx)
    }

    override func viewDidMoveToWindow() {
        super.viewDidMoveToWindow()
        guard let s = surface?.handle else { return }
        let scale = window?.backingScaleFactor ?? 2.0
        ghostty_surface_set_content_scale(s, scale, scale)
    }
}
```

- [ ] **Step 1: Create GhosttySurface.swift**

Write `/Users/fabio/Documents/playground/Agentic-IDE/AgenticIDE/Terminal/GhosttySurface.swift`:

```swift
import Foundation
import AppKit

/// Wraps one ghostty_surface_t. Created bound to a GhosttyNSView; freed on deinit.
@MainActor
final class GhosttySurface {
    private(set) var handle: OpaquePointer?

    /// - Parameters:
    ///   - view: the NSView whose CAMetalLayer Ghostty will draw into
    ///   - workingDirectory: cwd for the spawned shell; nil = $HOME
    ///   - command: shell command to run; nil = user's default login shell
    init?(view: GhosttyNSView, workingDirectory: String? = nil, command: String? = nil) {
        guard let app = GhosttyApp.shared.app else {
            print("GhosttySurface: GhosttyApp not bootstrapped")
            return nil
        }

        var surfaceConfig = ghostty_surface_config_new()
        surfaceConfig.platform_tag = GHOSTTY_PLATFORM_MACOS
        surfaceConfig.platform = ghostty_platform_u(macos: ghostty_platform_macos_s(
            nsview: Unmanaged.passUnretained(view).toOpaque()
        ))
        surfaceConfig.userdata = Unmanaged.passUnretained(self).toOpaque()
        surfaceConfig.scale_factor = view.window?.backingScaleFactor ?? 2.0

        // Attach optional command + working directory inside CString lifetimes
        let result: OpaquePointer? = withOptionalCString(command) { cCommand in
            surfaceConfig.command = cCommand
            return withOptionalCString(workingDirectory) { cCwd in
                surfaceConfig.working_directory = cCwd
                return ghostty_surface_new(app, &surfaceConfig)
            }
        }

        guard let surface = result else {
            print("ghostty_surface_new returned nil")
            return nil
        }

        self.handle = surface

        // Ghostty wants size + scale set immediately after creation
        let bounds = view.bounds
        let scale = view.window?.backingScaleFactor ?? 2.0
        ghostty_surface_set_content_scale(surface, scale, scale)
        ghostty_surface_set_size(surface, UInt32(bounds.width * scale), UInt32(bounds.height * scale))
        ghostty_surface_refresh(surface)
    }

    deinit {
        guard let surface = handle else { return }
        handle = nil
        // Free on the next main-loop tick so any in-flight callbacks finish first.
        // This mirrors cmux's deferred-free pattern.
        Task { @MainActor in
            ghostty_surface_free(surface)
        }
    }
}

/// Helper: turn an optional Swift String into a C string for the duration of `body`.
@MainActor
@inline(__always)
func withOptionalCString<R>(_ s: String?, _ body: (UnsafePointer<CChar>?) -> R) -> R {
    if let s {
        return s.withCString { body($0) }
    } else {
        return body(nil)
    }
}
```

```swift
import SwiftUI
import AppKit

struct GhosttySurfaceView: NSViewRepresentable {
    /// Working directory for the spawned shell. nil = $HOME.
    let workingDirectory: String?
    /// Command to run. nil = default login shell.
    let command: String?

    init(workingDirectory: String? = nil, command: String? = nil) {
        self.workingDirectory = workingDirectory
        self.command = command
    }

    func makeCoordinator() -> Coordinator { Coordinator() }

    func makeNSView(context: Context) -> GhosttyNSView {
        let view = GhosttyNSView(frame: .zero)
        // Defer surface creation until the view has a window (needed for scale_factor)
        DispatchQueue.main.async {
            guard view.window != nil else {
                // Will retry in viewDidMoveToWindow path; simplest: fire again
                self.makeSurfaceWhenReady(view: view, coordinator: context.coordinator)
                return
            }
            self.makeSurface(view: view, coordinator: context.coordinator)
        }
        return view
    }

    func updateNSView(_ nsView: GhosttyNSView, context: Context) {
        // Nothing to update for v1: surface lives for view lifetime.
    }

    static func dismantleNSView(_ nsView: GhosttyNSView, coordinator: Coordinator) {
        coordinator.surface = nil  // triggers GhosttySurface deinit -> ghostty_surface_free
    }

    @MainActor
    private func makeSurfaceWhenReady(view: GhosttyNSView, coordinator: Coordinator) {
        if view.window != nil {
            makeSurface(view: view, coordinator: coordinator)
        } else {
            DispatchQueue.main.asyncAfter(deadline: .now() + .milliseconds(16)) {
                self.makeSurfaceWhenReady(view: view, coordinator: coordinator)
            }
        }
    }

    @MainActor
    private func makeSurface(view: GhosttyNSView, coordinator: Coordinator) {
        guard coordinator.surface == nil else { return }
        guard let surface = GhosttySurface(view: view, workingDirectory: workingDirectory, command: command) else {
            print("GhosttySurfaceView: failed to create surface")
            return
        }
        view.surface = surface
        coordinator.surface = surface
        view.window?.makeFirstResponder(view)
    }

    final class Coordinator {
        var surface: GhosttySurface?
    }
}
```

```swift
final class GhosttyNSView: NSView, NSTextInputClient {
    weak var surface: GhosttySurface?

    override init(frame frameRect: NSRect) {
        super.init(frame: frameRect)
        wantsLayer = true
        layer?.masksToBounds = true
    }

    required init?(coder: NSCoder) { fatalError("init(coder:) not used") }

    override func makeBackingLayer() -> CALayer {
        let metalLayer = CAMetalLayer()
        metalLayer.pixelFormat = .bgra8Unorm
        metalLayer.isOpaque = false
        metalLayer.framebufferOnly = false
        return metalLayer
    }

    override var acceptsFirstResponder: Bool { true }

    override func becomeFirstResponder() -> Bool {
        let ok = super.becomeFirstResponder()
        if ok, let s = surface?.handle {
            ghostty_surface_set_focus(s, true)
        }
        return ok
    }

    override func resignFirstResponder() -> Bool {
        let ok = super.resignFirstResponder()
        if ok, let s = surface?.handle {
            ghostty_surface_set_focus(s, false)
        }
        return ok
    }

    override func setFrameSize(_ newSize: NSSize) {
        super.setFrameSize(newSize)
        guard let s = surface?.handle else { return }
        let scale = window?.backingScaleFactor ?? 2.0
        ghostty_surface_set_content_scale(s, scale, scale)
        ghostty_surface_set_size(s, UInt32(newSize.width * scale), UInt32(newSize.height * scale))
    }

    override func viewDidMoveToWindow() {
        super.viewDidMoveToWindow()
        guard let s = surface?.handle else { return }
        let scale = window?.backingScaleFactor ?? 2.0
        ghostty_surface_set_content_scale(s, scale, scale)
    }

    // MARK: - Keyboard

    override func keyDown(with event: NSEvent) {
        // Let the input system run first (it may call insertText: for us).
        interpretKeyEvents([event])

        // For special keys (arrows, escape, function keys, ctrl combos),
        // forward as a raw key event. cmux uses a populated ghostty_input_key_s;
        // the simplest viable subset is to send the event's `characters` via
        // ghostty_surface_text when the key has no modifiers and produces text,
        // and to fall through for everything else.
        //
        // Plan A v1: rely on interpretKeyEvents → insertText for normal typing,
        // and pass control/escape/arrows through ghostty_surface_text using
        // their raw character value.
        guard let s = surface?.handle else { return }
        if event.modifierFlags.contains(.control),
           let chars = event.charactersIgnoringModifiers, !chars.isEmpty {
            // Translate ctrl+letter to its control byte (e.g. ctrl+c -> 0x03)
            for scalar in chars.unicodeScalars {
                if scalar.value >= 0x40 && scalar.value <= 0x7F {
                    var byte: UInt8 = UInt8(scalar.value & 0x1F)
                    ghostty_surface_text(s, &byte, 1)
                }
            }
        }
    }

    // MARK: - NSTextInputClient (typed characters land here)

    func insertText(_ string: Any, replacementRange: NSRange) {
        guard let s = surface?.handle else { return }
        let str: String
        if let attr = string as? NSAttributedString { str = attr.string }
        else if let plain = string as? String { str = plain }
        else { return }

        var bytes = Array(str.utf8)
        bytes.withUnsafeMutableBufferPointer { buf in
            ghostty_surface_text(s, buf.baseAddress, UInt(buf.count))
        }
    }

    func doCommand(by selector: Selector) {
        // For arrows / escape / return / tab — translate to bytes Ghostty understands.
        guard let s = surface?.handle else { return }
        let bytes: [UInt8]?
        switch selector {
        case #selector(NSStandardKeyBindingResponding.insertNewline(_:)):
            bytes = [0x0D]
        case #selector(NSStandardKeyBindingResponding.insertTab(_:)):
            bytes = [0x09]
        case #selector(NSStandardKeyBindingResponding.cancelOperation(_:)):
            bytes = [0x1B]
        case #selector(NSStandardKeyBindingResponding.deleteBackward(_:)):
            bytes = [0x7F]
        case #selector(NSStandardKeyBindingResponding.deleteForward(_:)):
            bytes = [0x1B, 0x5B, 0x33, 0x7E] // ESC [ 3 ~
        case #selector(NSStandardKeyBindingResponding.moveUp(_:)):
            bytes = [0x1B, 0x5B, 0x41]
        case #selector(NSStandardKeyBindingResponding.moveDown(_:)):
            bytes = [0x1B, 0x5B, 0x42]
        case #selector(NSStandardKeyBindingResponding.moveRight(_:)):
            bytes = [0x1B, 0x5B, 0x43]
        case #selector(NSStandardKeyBindingResponding.moveLeft(_:)):
            bytes = [0x1B, 0x5B, 0x44]
        default:
            bytes = nil
        }
        if var b = bytes {
            b.withUnsafeMutableBufferPointer { buf in
                ghostty_surface_text(s, buf.baseAddress, UInt(buf.count))
            }
        }
    }

    // Required NSTextInputClient stubs (no IME support in Plan A)
    func setMarkedText(_ string: Any, selectedRange: NSRange, replacementRange: NSRange) {}
    func unmarkText() {}
    func selectedRange() -> NSRange { NSRange(location: NSNotFound, length: 0) }
    func markedRange() -> NSRange { NSRange(location: NSNotFound, length: 0) }
    func hasMarkedText() -> Bool { false }
    func attributedSubstring(forProposedRange range: NSRange, actualRange: NSRangePointer?) -> NSAttributedString? { nil }
    func validAttributesForMarkedText() -> [NSAttributedString.Key] { [] }
    func firstRect(forCharacterRange range: NSRange, actualRange: NSRangePointer?) -> NSRect { .zero }
    func characterIndex(for point: NSPoint) -> Int { NSNotFound }
}
```
